# Flow B — Learning Topics (local preview)

Files added:

- `index.html` — main landing page and navigation
- `credit-simulator.html` — credit score slider, loan amount input, basic APR estimation
- `guides.html` — responsive video embed and article cards
- `money-tips.html` — carousel and tip cards with badges

How to preview locally in VS Code with Microsoft's Live Preview extension:

1. Install the **Live Preview** extension by Microsoft (if not installed).
2. Open this workspace in VS Code.
3. Open `index.html` in the editor.
4. Press `Ctrl+Shift+P` and run `Live Preview: Show Preview` or click the Live Preview controls in the editor title bar. Alternatively, right-click the editor and choose **Open with Live Preview**.
5. Use the navigation links or the offcanvas menu to open `credit-simulator.html`, `guides.html`, or `money-tips.html`.

If you prefer Live Server (Ritwick Dey), you can use it instead; open the workspace and click "Go Live".
